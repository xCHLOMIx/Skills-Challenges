Skills Challenges API Documentation

Base URL

http://localhost:1234

Authentication

- Participants require JWT token in cookies named 'jwt'
- Admins require JWT token in cookies named 'jwtAdmin'
- Tokens expire after 72 hours
- Tokens are automatically set after successful login

Authentication Endpoints

POST /auth/signup

Register a new participant

{
    "name": "string",
    "email": "string",
    "phone": "number",
    "title": "string",
    "password": "string"
}
POST /auth/login

Login for participants

{
    "email": "string",
    "password": "string"
}
POST /auth/signupAdmin

Register a new admin

{
    "name": "string",
    "email": "string",
    "phone": "number",
    "password": "string"
}
POST /auth/loginAdmin

Login for admins

{
    "email": "string",
    "password": "string"
}

Challenge Endpoints

GET /challenges

Get all challenges (Requires participant authentication)

GET /challenges/:id

Get a specific challenge (Requires participant authentication)

POST /challenges

Create a new challenge (Requires admin authentication)

{
    "title": "string",
    "deadline": "date",
    "duration": "number",
    "moneyPrize": "number",
    "contactEmail": "string",
    "projectDescription": "string",
    "projectBrief": "string",
    "projectTasks": "string",
    "skills": ["string"],
    "seniorityLevel": "string"
}
PUT /challenges/:id

Update a challenge (Requires admin authentication)

{
    "title": "string",
    "deadline": "date",
    "duration": "number",
    "moneyPrize": "number",
    "contactEmail": "string",
    "projectDescription": "string",
    "projectBrief": "string",
    "projectTasks": "string",
    "skills": ["string"],
    "seniorityLevel": "string"
}
DELETE /challenges/:id

Delete a challenge (Requires admin authentication)

POST /challenges/join

Join a challenge (Requires participant authentication)

{
    "userId": "string",
    "challengeId": "string"
}

Models

Challenge Model

{
    title: String,
    deadline: Date,
    duration: Number,
    moneyPrize: Number,
    contactEmail: String,
    projectDescription: String,
    projectBrief: String,
    projectTasks: String,
    skills: [String],
    seniorityLevel: String,
    status: Boolean
}

Participant Model

{
    name: String,
    email: String,

    phone: Number,
    challengeId: ObjectId,
